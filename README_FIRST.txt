# searchbar-React
A autocomplete search bar using ReactJS

SETUP and RUN React app
------------------------
I am assuming that you have Node.js (node_module package) installed (If not then use npm install), and you are using Visual Studio Code.

Open terminal or Command Prompt and follow the given steps:

** To install 'create-react-app' generator
npm install -g create-react-app

** Create a new React application (new application)
create-react-app <app-name>

** Opening in Visual Studio Code
cd <app-name> & code .

** In Visual Studio Code
For executing the app (npm start) you need
npm install react-scripts -g

&

** For AJAX call you need the axios module
npm install axios
